MobaXterm工具(1)——网络连接

Share

SSH连接

选项介绍

  • Execute command

连接后，将在远程服务器上执行指定的命令。

您可以指定多个命令并用“;”分隔它们要么 ”＆＆”。

例如：

pwd; uname -a

测试一下

  • X11-Forwarding

X11-Forwarding（通过SSH通道自动显示重定向）

X11-Forwarding是一种允许X11协议通过现有SSH通道路由的机制。该解决方案的主要优点是：

  1. X11协议将被加密

  2. 可以压缩X11协议

  3. 无需为X11设置新的通信通道（以及在防火墙中打开通信端口）：网络数据包通过现有的SSH连接

  • Compression

压缩通过SSH通道传输的数据,这对慢速连接（WAN）很有用

  • Remote environment

在新窗口中从远程服务器启动完整桌面。例如，在Raspberry Pi设备上，您可以启动LXDE。

  • Use private key

指定用于无密码登录的私钥。

可以使用以下命令生成新密钥：

ssh-keygen -t rsa

要么

ssh-keygen -t dsa

还可以使用嵌入式MobaKeyGen应用程序（可在“工具”菜单中找到）创建和管理SSH密钥。

  • Adapt locales on remote server

调整远程服务器上的区域设置

尝试将本地语言设置发送到远程服务器：

这允许远程服务器知道本地使用的语言和字符集设置。

注意：远程服务器必须支持这些语言设置。

  • SSH-browser type

显示图形化SSH浏览器，允许您浏览远程服务器内容。

可用设置：

无：禁用SSH浏览器

SFTP协议：启用SSH浏览器并使用SFTP协议

SCP（增强速度）：启用SSH浏览器并使用最新的SCP协议实现最大传输速度

SCP（正常速度）：启用SSH浏览器并使用旧的SCP协议实现传统支持

注意：与FTP协议不同，SSH浏览器使用安全的加密SSH通道进行传输。

RDP

  • Redirect drives

重定向驱动器

在远程服务器上显示本地驱动器

  • Enhanced graphics

增强的图形

使用远程桌面的增强设置

  • Redirect clipboard

重定向剪贴板

允许从/到RDP选项卡进行复制/粘贴

  • Forward keyboard shortcuts

转发键盘快捷键

键盘快捷方式将转发到远程桌面，而不是由本地Windows环境处理

FTP

  • Passive mode

被动模式

建议使用此模式，尤其是在防火墙后面的情况下。

在此模式下，FTP客户端启动2个连接（控制和数据），而在活动模式下，控制连接由客户端启动，服务器启动数据连接。

  • ASCII mode

ASCII模式

传输过程中文本文件将转换为Unix / Dos格式。

警告：此功能不适用于二进制文件！

  • Encrypt data channel

加密数据通道

如果选中此选项，则将加密用于数据传输的通道（文件，目录列表），否则仅加密命令通道。

SFTP

  • ASCII mode

ASCII模式

传输过程中文本文件将转换为Unix / Dos格式。

警告：此功能不适用于二进制文件！

  • Preserve files dates

保留文件日期

如果要在通过SFTP会话传输文件日期时保留文件日期，请选中此设置
